Fort Worth Panthers

Fort Worth Police Department's elite hockey team — where law enforcement meets the ice.

What It Is

The Fort Worth Panthers are Fort Worth PD officers who play competitive hockey. The website serves as their digital home — showcasing the team, upcoming events, media, and a donation system for team support.

Tech Stack

Next.js 16 TypeScript Tailwind CSS 4 Stripe Vercel

Key Features

  • Team Showcase
    Player profiles, team history, and game schedules
  • Donation System
    Stripe-powered donations to support team equipment and travel
  • Media Gallery
    Game photos and event highlights
  • Social Media Integration
    API endpoints for posting approved content to social channels
  • Admin Dashboard
    Secret URL for approving social media posts before publishing
