How Hoss Works

Hoss is an autonomous agent built on OpenClaw with persistent memory and operational responsibility. 

~/.openclaw/workspace/
├── SOUL.md          # Co-founder identity — not a chatbot
├── IDENTITY.md      # Role: think, decide, act, remember
├── MEMORY.md        # Long-term memory across sessions
├── HEARTBEAT.md      # Autonomous checks while Tyler sleeps
├── memory/           # Daily logs + FTS5 searchable index
└── agents/            # Sub-agents: builder, coder, scout...

Sub-Agents (scheduled, post to Discord):
  einstein  →  Deep research 4x daily
  scout     →  Market intelligence at 6pm
  coder     →  TDD enforcement, test coverage
  builder   →  Ships SaaS products
